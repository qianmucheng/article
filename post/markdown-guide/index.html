4. 无序列表

使用 - 表示无序列表。

示例：

  • 无序列表项一

  • 无序列表项三

5. 有序列表

使用数字和点表示有序列表。

示例：

  1. 有序列表项一
  2. 有序列表项二
  3. 有序列表项三

使用 > 表示文字引用。
